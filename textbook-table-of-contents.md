# Original Textbook

This course includes an original textbook written for the flow of WEB 1430.

The textbook is maintained alongside the live course materials. Recent alignment updates strengthened the chapters that support:

- API viability checks for browser-based projects
- safe browser-storage use with fictional/demo data
- lightweight project documentation through `README.md`
- deployment QA, attribution, and pre-submission verification

- Chapter 1: Thinking in the Browser
- Chapter 2: HTML, CSS, and Progressive Enhancement
- Chapter 3: JavaScript Syntax, Values, and Expressions
- Chapter 4: Decisions, Loops, and Reusable Logic
- Chapter 5: Modeling Information in JavaScript
- Chapter 6: The Document Object Model
- Chapter 7: Event-Driven Interfaces and Forms
- Chapter 8: Design Systems and Small Front-End Architecture
- Chapter 9: Fetch, JSON, and Remote Data
- Chapter 10: Storage, Preferences, and State
- Chapter 11: Modules, NPM, and Vite
- Chapter 12: Introductory Component-Based Development
- Chapter 13: Accessibility Synthesis
- Chapter 14: Performance, Testing, and Deployment
